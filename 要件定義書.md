# 素数マスター — 要件定義書

**作成日：** 2026-04-03  
**バージョン：** 1.0

---

## プロジェクト概要

「素数マスター」は、素数を楽しく覚えるためのタイムアタック型 Web クイズアプリです。  
画面に表示された数字が**素数かどうかを素早く判定**し、60秒以内の正解数を競います。  
PC・スマートフォン両対応のレスポンシブ設計とし、難易度別ランキングを搭載します。

---

## 機能要件

### 1. トップ画面（ホーム）

| 項目 | 内容 |
|------|------|
| アプリ名 | 素数マスター（Prime Master） |
| 難易度選択 | 2桁 / 3桁 / 4桁 の3段階 |
| 遷移 | 難易度を選択してゲーム開始ボタンで即スタート |
| ランキング閲覧 | 各難易度のランキングをトップ画面から参照可能 |

---

### 2. ゲーム画面

#### 2-1. ゲームフロー

```
[ホーム] → 難易度選択 → [3秒カウントダウン] → [ゲーム中] → [結果画面] → [ホームへ戻る]
```

#### 2-2. ゲーム中の動作

| 項目 | 内容 |
|------|------|
| 出題形式 | 選択難易度の桁数の整数をランダム表示 |
| 桁数の定義 | 2桁：10〜99、3桁：100〜999、4桁：1000〜9999 |
| 回答方法 | 「✅ 素数」「❌ 素数でない」の2ボタン押下 |
| タイムリミット | 60秒（カウントダウンタイマー表示） |
| 正誤判定 | 即時フィードバック（アニメーション＋色で明示） |
| 次の問題 | 正誤判定後、自動的に次の数字へ遷移 |
| スコア表示 | 残り時間・正解数・不正解数をリアルタイム表示 |
| 問題数上限 | 60秒以内に解ける数（上限なし） |

#### 2-3. 素数生成ロジック

- 対象桁数の範囲内の数字をランダム生成
- 素数と非素数が**おおよそ半々**になるよう出題
- 同じ数字が連続して出ないよう制御

---

### 3. 結果画面

| 項目 | 内容 |
|------|------|
| 表示内容 | 正解数・不正解数・正答率・難易度 |
| ランキング登録 | 名前入力（最大10文字）→「登録する」ボタン |
| ランキング表示 | 登録後、当該難易度のランキングを即時表示 |
| 再挑戦 | 同じ難易度で再スタートするボタン |
| ホームへ | トップ画面へ戻るボタン |

---

### 4. ランキング機能

| 項目 | 内容 |
|------|------|
| 保存方式 | ブラウザの `localStorage`（サーバー不要） |
| ランキング区分 | 難易度ごとに独立管理（2桁・3桁・4桁それぞれ別） |
| 表示件数 | 上位10件を表示 |
| ランキング項目 | 順位 / 名前 / スコア（正解数）/ 日時 |
| 同スコアの扱い | 正解数が同じ場合は登録日時が早い方を上位とする |

## 非機能要件

### デザイン・UI

| 項目 | 内容 |
|------|------|
| テーマ | ダークモード基調、宇宙・数学をイメージしたスタイリッシュなデザイン |
| カラーパレット | 深い紺〜パープル系のグラデーション、アクセントは金色・シアン |
| フォント | Google Fonts（Outfit, Orbitron） |
| アニメーション | 正解時：緑のフラッシュ＋パーティクル、不正解時：赤の振動エフェクト |
| タイマー | 残り10秒以下で赤く点滅し焦燥感を演出 |

### レスポンシブ対応

| ブレークポイント | レイアウト |
|----------------|-----------|
| PC（768px〜） | ゲームエリアを中央寄せ、適度な余白 |
| スマホ（〜767px） | 縦スクロールなしで1画面に収まるUI、ボタン大きめ |

### 技術スタック

| 項目 | 内容 |
|------|------|
| 実装 | HTML + CSS + Vanilla JavaScript（3ファイル構成） |
| データ永続化 | `localStorage`（インストール・サーバー不要） |
| 外部依存 | Google Fonts のみ（CDN） |
